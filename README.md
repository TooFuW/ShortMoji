## How to use it
Type a shortcut from the list [HERE](https://github.com/TooFuW/ShortMoji?tab=readme-ov-file#keyboard-shortcuts), press `space` and it will automatically be replaced by the corresponding emoji, it's that simple ! ğŸ¥³

*The program may not always work in system applications (e.g. Task Manager) because Windows blocks keyboard access at that time*

If you want to close the program, just press your `escape` key twice !
